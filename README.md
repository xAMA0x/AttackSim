Simulateur d'attaques cryptographiques - Projet ESGI

## Structure

- `src/core/` : Moteur principal et UI
- `src/attacks/symmetric/` : Attaques DES, AES
- `src/attacks/asymmetric/` : Attaques RSA, ECC
- `src/attacks/special/` : Attaques Timing
- `reports/` : Exports PNG des graphiques
